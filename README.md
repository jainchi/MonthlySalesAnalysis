**# 📊 Monthly Sales Analysis (Python Project)

## 🚀 Overview

This project analyzes 9 months of sales data using Python (Pandas).
The goal was to clean raw data, perform analysis, and extract meaningful business insights.

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* VS Code
* Git & GitHub

---

## 📂 Dataset

* Multiple monthly CSV files
* Contains order details like product, price, quantity, date, and address

---

## 🔧 Process

### 1. Data Loading

* Loaded multiple CSV files using `glob`
* Combined them into a single DataFrame using `concat`

### 2. Data Cleaning

* Removed duplicate header rows
* Handled missing values
* Converted data types (datetime, numeric)

### 3. Feature Engineering

* Extracted Month from Order Date
* Extracted Day and City from data
* Created `Sales` column (Quantity × Price)

---

## 📊 Key Insights

* 📅 **December** had the highest sales

* 📈 Sales peaked during year-end months (Oct–Dec)

* 📦 **iPhone & MacBook Pro** generated highest revenue

* 🔋 **Batteries & cables** were most sold (high volume, low price)

* 🌍 **San Francisco** had highest sales

* 🏙️ Metropolitan cities performed better

* 💰 Price range:

  * Highest: 1700
  * Lowest: 2.99

---

## 🧠 Conclusion

* Revenue is driven by high-value products
* Volume is driven by low-cost accessories
* Sales are higher in big cities and festive periods
